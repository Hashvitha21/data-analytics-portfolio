# 1. Student Performance Analysis

## Project Overview
SQL project analyzing student performance data by transforming raw datasets, building dimension tables, and performing exploratory analysis to identify factors influencing academic outcomes.

The project examines the impact of:
- Study habits
- Tutoring
- Attendance
- Parental support
- Extracurricular activities

on student grades and GPA performance.

---

## Key SQL Operations
- Data cleaning and formatting
- Encoded value conversion
- Data modeling using dimension tables
- Foreign key relationships
- Aggregations and analytical queries
- CASE statements and joins

---

## Analysis Performed
- Tutoring vs Performance
- Study Time vs GPA
- Absences vs Academic Performance
- Parental Support vs Grades
- Extracurricular Activities vs GPA

---

## Tools Used
- MySQL
- SQL
- Data Modeling
- EDA

---

## Key Insights
- Students with tutoring support showed better academic performance.
- Increased study time positively impacted GPA.
- Higher absenteeism correlated with lower grades.
- Strong parental support improved overall student outcomes.

# 2. Pharmaceutical Sales Analysis

## Project Overview
SQL project focused on cleaning, standardizing, transforming, and analyzing pharmaceutical sales data to generate meaningful business insights.

The project involved handling inconsistent data, removing duplicates, standardizing records, and performing revenue analysis using advanced SQL techniques.

---

## Key SQL Operations
- Data cleaning and standardization
- Missing value handling
- Type conversion
- Deduplication using ROW_NUMBER()
- Window functions and CTEs
- Outlier detection using IQR method

---

## Analysis Performed
- Top-selling medicines
- Revenue analysis by medicine
- City-wise sales performance
- Doctor-wise revenue analysis
- Monthly revenue trends
- Category-wise sales analysis
- Outlier transaction detection

---

## Tools Used
- MySQL
- SQL
- Window Functions
- EDA

---

## Key Insights
- Certain medicines generated significantly higher revenue.
- Sales performance varied across cities and categories.
- Data standardization improved reporting accuracy.
- Outlier analysis helped identify unusual transactions.

# 4. Profit Performance Dashboard

## Dashboard Overview
Business intelligence dashboard focused on brand-wise profit analysis and revenue performance across countries and product categories.

The dashboard analyzes:
- Monthly profit trends
- Brand contribution
- Country-wise profitability
- Product category performance
- Revenue and margin analysis

---

## Dashboard Highlights
- Canada and the UK generated the highest profits.
- Profit contribution remained balanced across brands.
- Praline and White Chocolate categories performed strongest.
- Monthly profitability trends remained stable throughout the year.
